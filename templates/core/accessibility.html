Acessibilidade

O BusFeed foi desenvolvido seguindo as diretrizes de acessibilidade WCAG 2.1, garantindo que todos possam usar nossos serviços independentemente de suas capacidades.

Recursos de Acessibilidade

Para Usuários com Deficiência Visual

  • Compatible com leitores de tela (NVDA, JAWS, VoiceOver)
  • Navegação por teclado em todas as funcionalidades
  • Alto contraste disponível
  • Textos alternativos em todas as imagens
  • Estrutura semântica clara

Para Usuários com Deficiência Motora

  • Áreas de clique aumentadas em botões
  • Navegação apenas por teclado
  • Tempo de sessão estendido
  • Atalhos de teclado disponíveis

Atalhos de Teclado

Atalho Função
Alt + 1 Ir para página inicial
Alt + 2 Ir para busca de rotas
Alt + 3 Ir para lista de paradas
Alt + M Ir para menu principal
Escape Fechar modais e menus

Conformidade e Padrões

O BusFeed está em conformidade com:

  • WCAG 2.1 Nível AA: Diretrizes de Acessibilidade para Conteúdo Web
  • Lei Brasileira de Inclusão (LBI): Lei 13.146/2015
  • eMAG: Modelo de Acessibilidade em Governo Eletrônico

Melhorias Contínuas

Trabalhamos constantemente para melhorar a acessibilidade. Se você encontrar alguma barreira ou tiver sugestões, entre em contato conosco.
