Introdução ao BusFeed

O que é o BusFeed?
O BusFeed é um sistema inteligente de transporte público desenvolvido especificamente para Brasília e o Distrito Federal. Ele permite consultar rotas de ônibus, visualizar paradas no mapa, verificar horários e encontrar a melhor forma de se locomover pela cidade usando transporte público.
O BusFeed é gratuito?
Sim! O BusFeed é completamente gratuito e foi desenvolvido para facilitar a vida dos usuários de transporte público do DF. Nosso objetivo é tornar a mobilidade urbana mais acessível e eficiente para todos.

Busca de Rotas

Como buscar rotas para um destino?
Na página inicial, digite seu destino no campo de busca (ex: "UnB", "Taguatinga", "Shopping"). O sistema mostrará todas as rotas que passam pelo local ou próximo a ele. Você também pode usar nomes de bairros, pontos de referência ou terminais.
Como interpretar as informações das rotas?
Cada rota mostra: número da linha, nome do trajeto, empresa responsável, tarifa e se possui acessibilidade para cadeirantes. Clique em "Ver Detalhes" para ver o trajeto completo com todas as paradas no mapa.

Usando os Mapas

Como navegar no mapa?
Use o mouse para arrastar o mapa, a roda do mouse ou os botões + e - para aproximar/afastar. Em dispositivos móveis, use gestos de toque. Clique nos marcadores das paradas para ver informações detalhadas.
O que significam as cores e símbolos no mapa?
Paradas simples: marcadores cinza pequenos
Terminais: marcadores vermelhos maiores
Estações de metrô: marcadores laranja
Rotas ativas: linhas coloridas conectando as paradas
Números nos marcadores: sequência de paradas na rota

Acessibilidade

Como encontrar rotas acessíveis?
Na página de rotas, use o filtro "Apenas acessíveis" para mostrar somente linhas com ônibus adaptados para cadeirantes. O ícone indica rotas e paradas acessíveis.
O site funciona com leitores de tela?
Sim! O BusFeed foi desenvolvido seguindo padrões de acessibilidade web (WCAG). É compatível com leitores de tela como NVDA, JAWS e VoiceOver. Use a tecla Tab para navegar pelos elementos e Enter para ativar links e botões.

Problemas Comuns

O mapa não está carregando
Verifique sua conexão com a internet. Se o problema persistir, tente recarregar a página (F5) ou limpar o cache do navegador. Em dispositivos móveis, certifique-se de que o JavaScript está habilitado.
Não encontrei minha rota
Tente diferentes termos de busca: nome do bairro, ponto de referência próximo, ou terminal mais próximo. Se ainda não encontrar, a rota pode não estar em nossa base de dados ainda. Entre em contato para reportar rotas faltantes.
Os horários estão incorretos
Os dados são atualizados regularmente, mas pode haver variações nos horários reais devido a trânsito, eventos ou mudanças operacionais. Sempre confirme os horários nos terminais ou com a empresa responsável pela linha.
