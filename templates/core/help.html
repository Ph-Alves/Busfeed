Primeiros Passos

O BusFeed é simples de usar:

  1. Na página inicial, você pode buscar rotas digitando o número da linha ou destino
  2. Use "Paradas Próximas" para encontrar paradas perto de você
  3. Consulte horários em tempo real clicando em qualquer rota
  4. Use os atalhos de teclado (Alt+1, Alt+2, Alt+3) para navegar rapidamente

Para encontrar paradas próximas:

  1. Clique em "Paradas Próximas" na página inicial
  2. Permita o acesso à sua localização quando solicitado
  3. O sistema mostrará as paradas mais próximas com distância
  4. Clique em qualquer parada para ver quais linhas passam por ela

Rotas e Horários

Você pode pesquisar rotas de várias formas:

  • Por número da linha: Digite "101" para encontrar a linha 101
  • Por destino: Digite "Taguatinga" para ver rotas que vão para lá
  • Por ponto de referência: Digite "Rodoviária" para ver rotas que passam lá

O sistema mostra sugestões conforme você digita.

  • Previsão real de chegada dos ônibus
  • Atrasos ou adiantamentos

Acessibilidade

O BusFeed oferece vários atalhos para navegação rápida:

  • Alt + 1 - Ir para página inicial
  • Alt + 2 - Focar no campo de busca
  • Alt + 3 - Ir para paradas
  • Alt + R - Ir para rotas
  • Alt + H - Ir para horários
  • Alt + M - Focar no menu principal

O BusFeed é totalmente compatível com leitores de tela:

  • NVDA (Windows)
  • JAWS (Windows)
  • VoiceOver (Mac/iOS)
  • TalkBack (Android)

Todas as informações são anunciadas claramente e a navegação é otimizada para uso apenas com teclado.

Problemas Técnicos

Se o BusFeed não está carregando:

  1. Verifique sua conexão com a internet
  2. Tente recarregar a página (F5 ou Ctrl+R)
  3. Limpe o cache do navegador
  4. Tente em modo anônimo/privado
  5. Verifique se não há bloqueadores de anúncio interferindo

Se o problema persistir, entre em contato conosco.

Se você encontrou informações incorretas:

  1. Anote exatamente qual informação está errada
  2. Inclua a linha, parada ou horário específico
  3. Use nosso formulário de contato
  4. Selecione "Dados incorretos" como categoria
  5. Descreva o problema detalhadamente

Nossa equipe investigará e corrigirá rapidamente.
